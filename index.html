Reminder

Keep sensitive data out of public repositories

GitHub Pages publishes static files publicly. Do not store API keys, passwords, tokens, or private company data in this repository.

In GitHub, open Settings -> Pages, choose "Deploy from a branch", then select the `main` branch and the `/ (root)` folder.
